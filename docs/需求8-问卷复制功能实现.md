# 需求8：问卷复制功能 - 实现总结

## 实现日期
2026-03-22

## 实现内容

### 1. 后端实现

#### 文件：`backend/src/controllers/surveyController.ts`
- 新增 `duplicateSurvey` 函数
- 功能：
  - 获取原问卷及其所有问题和选项
  - 创建新问卷，标题 = 原标题 + "（副本）"
  - 复制所有题目（questions）和选项
  - 新问卷状态设为 "draft"（草稿）
  - 不复制填写数据（responses）
  - 返回新创建的问卷（含问题和选项）

#### 文件：`backend/src/routes/surveyRoutes.ts`
- 新增路由：`POST /api/surveys/:id/duplicate`
- 需要身份验证（authMiddleware）

### 2. 前端实现

#### 文件：`frontend/src/api/survey.ts`
- 新增 API 方法：`duplicateSurvey(id: string)`
- 调用后端接口：`POST /surveys/${id}/duplicate`

#### 文件：`frontend/src/views/SurveyList.vue`
- 在问卷卡片上添加"复制"按钮
- 位置：在"编辑"按钮旁边
- 点击后：
  1. 调用复制接口
  2. 显示成功提示
  3. 跳转到新问卷的编辑页面

## 技术实现细节

### 复制逻辑
1. 使用 `findByPk` 配合 `include` 获取完整的问卷数据（含问题和选项）
2. 创建新问卷记录，标题追加"（副本）"，状态设为 'draft'
3. 遍历原问卷的所有问题，逐个创建新问题记录
4. 对每个问题的选项也进行复制
5. 返回新创建的完整问卷数据

### 权限控制
- 需要用户登录（authMiddleware）
- 新问卷的 creatorId 为当前登录用户

### 数据库操作
- 不使用事务（可根据需要添加）
- CASCADE 删除设置确保数据一致性
